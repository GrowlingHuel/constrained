# Tether

A typing constraint engine for writers.

Apply rules to your writing sessions — word length limits, alphabet cycling,
no repeated words, time sprints, pangram challenges — and measure your
compliance, vocabulary richness, and readability in real time.

## Run from source

Requires Python 3.8+ and tkinter.
```bash
python3 constrained.py
```

Optional (for linguistic features):
```bash
pip3 install spacy textstat
python3 -m spacy download en_core_web_sm
```
